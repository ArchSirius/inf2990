Description détaillée

Classe permettant le dessin d'un modèle 3D par VBO.

Note
Précondition : prends pour acquis que le pointeur vers le modèle 3D reste valide durant toute la durée de vie du storage. Le modèle est considéré "constant", c'est-à-dire que ses meshes ne sont pas modifiés et qu'il n'est pas relâché lors de l'appel au dessin. Une modification à la géométrie du modèle nécessite de libérer le VBO et le recréer.
