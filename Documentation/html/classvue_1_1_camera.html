Description détaillée

Classe représentant une caméra dans le monde en 3D.

Cette camera encapsule les différentes opérations qu'il est possible de faire pour déplacer le point de vue de l'observateur à l'intérieur de la scène en 3D.

Auteur
Martin Bisson
Date
2006-12-15
