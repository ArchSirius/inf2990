Le livrable 3 compilé se trouve dans le répertoire Livrable 3.
L'exécutable est InterfaceGraphique.exe.
Le répertoire Livrable 3 est autonome, il n'a pas besoin de ses répertoires voisins pour fonctionner.
La zone de simulation par défaut est zones\Default.scene. Elle contient trois poteaux gris, deux petits murs rouges et des lignes noires.
La documentation à jour se trouve dans le répertoire Documentation.
À des fins pratiques, il est possible d'activer l'enregistrement des informations de la console dans des fichiers journaux (log files), dans le répertoire logs\.
