Le livrable compilé se trouve dans le répertoire Exe.
L'exécutable est Exe\InterfaceGraphique.exe
Le répertoire Exe est autonome, il n'a pas besoin de ses répertoires voisin pour fonctionner.
La zone de simulation par défaut est Exe\zones\Default.scene. Elle contient un poteau gris, un petit mur rouge et des lignes noires.
La documentation à jour se trouve dans le répertoire Exe\Documentation.
